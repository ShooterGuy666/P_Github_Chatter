**Installatie**

- Stap 1. Download de github repository.
- Stap 2. Installeer nodeJS.
- Stap 3. Open een terminal.
- Stap 4. Voer de volgende commands in:
- Stap 4.1. npm i
- Stap 4.2. npm start
